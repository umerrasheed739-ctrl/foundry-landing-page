# Foundry — Digital Craftsmanship Studio Landing Page

A semantic, accessible, and high-performance landing page built with vanilla web technologies (HTML5, CSS3, and JavaScript). Designed with a mobile-first responsive architecture and a custom 2025 design palette.

## 🚀 Repository Link

- **GitHub Repo:** [Foundry Landing Page](https://github.com/umerrasheed739-ctrl/foundry-landing-page)

---

## ✨ Features

- **Semantic HTML5:** Built using proper structural landmarks (`header`, `nav`, `main`, `section`, `article`, `footer`) for accessibility and SEO.
- **Responsive Architecture:** Built mobile-first using CSS Grid for macro layouts and Flexbox for micro components.
- **Custom Design Tokens:** Utilizes CSS Custom Properties (`--mocha`, `--ethereal-blue`, `--moonlit-grey`) for consistent spacing, typography, and color schemes.
- **Dark / Light Theme Toggle:** Fully functional JavaScript theme switcher with local state persistence and CSS variable overrides.
- **Accessible UI:** Includes skip navigation link, structured heading hierarchy, custom focus states, and screen-reader considerations (`aria-*` attributes).
- **Interactive Form:** Vanilla JavaScript client-side form validation with accessible error messaging.

---

## 🛠️ Tech Stack

- **HTML5** — Semantic Structure & Accessibility
- **CSS3** — Custom Properties, Flexbox, CSS Grid, Responsive Media Queries
- **JavaScript (Vanilla)** — DOM Manipulation, Theme Toggling & Form Validation
